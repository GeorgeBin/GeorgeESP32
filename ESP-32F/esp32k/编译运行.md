# 编译运行



### 查看端口

```bash
ls /dev/cu.*
```

当前板子端口：

```bash
/dev/cu.usbserial-142101
```



## 使用命令行

因为在 `~/.zshrc` 里配置了 alias，则使用：

```
get_idf_554
idf.py --version
```



### 编译

```bash
idf.py build
```

### 烧录、看日志

```bash
idf.py -p /dev/cu.usbserial-142101 flash monitor
```

### 退出 monitor

```bash
Ctrl + ]
```
