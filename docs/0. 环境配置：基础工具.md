# 环境配置：基础工具



先安装 Xcode 命令行工具：

```bash
xcode-select --install
```

然后安装 Homebrew。已经装过可以跳过：

```bash
brew --version
```

安装基础依赖：

```bash
brew install git cmake ninja dfu-util python ccache
```

确认版本：

```bash
git --version
python3 --version
cmake --version
ninja --version
```

建议不要使用 Anaconda / Miniconda 的 Python 作为 ESP-IDF 主 Python，避免后续 `idf.py`、虚拟环境、依赖包版本混乱。
