# 环境配置：安装 ESP-IDF：压缩包



### 安装 ESP-IDF：v5.5.4

#### 下载压缩包方式

```bash
# 下载
wget -c https://dl.espressif.com/github_assets/espressif/esp-idf/releases/download/v5.5.4/esp-idf-v5.5.4.zip

# 解压到 /Users/GeorgeSystem/Develop/Environment/esp/esp-idf-v5.5.4
# 切换到对应目录
cd /Users/GeorgeSystem/Develop/Environment/esp/esp-idf-v5.5.4

# 安装
./install.sh esp32
```



> 其“安装”步骤，还是会在 ~/.espressif 目录下，所以如果之前用其他方式下载了，可以先清除此目录，再安装。
