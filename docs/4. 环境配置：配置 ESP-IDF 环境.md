# 环境配置：配置 ESP-IDF 环境

使用：ESP-IDF v5.5.4



### 配置 ESP-IDF 环境变量

每次打开新终端后执行：

```bash
# eim 方式
. /Users/george/.espressif/tools/activate_idf_v5.5.4.sh

# zip 方式
. /Users/GeorgeSystem/Develop/Environment/esp/esp-idf-v5.5.4/export.sh
```

验证：

```
idf.py --version
echo $IDF_PATH
```



### 在 `~/.zshrc` 里加一个 alias（这里使用 zip 方式）

```bash
open ~/.zshrc
```

填入内容：

```
alias get_idf_554='. /Users/GeorgeSystem/Develop/Environment/esp/esp-idf-v5.5.4/export.sh'
```

以后使用时：

```
get_idf_554
idf.py --version
```

> 以后如果同时安装 v5.5、v6.0，也不容易混乱。
